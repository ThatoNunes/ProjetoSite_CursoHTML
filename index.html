Tags

Voltar

TAGS:são marcadores usados ​​para delimitar seções de um documento HTML.

Tem um significado semântico e define um bloco de conteúdo que pode ser manipulado pelo navegador.

São extremamente úteis porque permitem que os navegadores interpretem e apresentem corretamente o conteúdo.

Além disso, também podem ser usadas para aplicar estilos a um documento HTML usando folhas de estilo em cascata (CSS).

  • Lista de Tags

    • Exemplos de Tags usada no curso:

    Elementos h1:

    Voltar

    representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.

    Destina-se a marcar o cabeçalho de uma seção ou da página como um todo e deve ter informações sober o conteundo da seção.

    Exemplo: 
                <body>
                 <header>
                 <h1>Título principal da página</h1>
                 </header>
                 //conteúdo da página
            </body>

    Elemento <p> ou Parágrafo

    Voltar

    Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada elemento <p>.

    Exemplo: 
                <p>Este é o primeiro parágrafo</p>
            <p>Este é o segundo parágrafo</p>

    Elemento <u> ou Sublinhado

    Voltar

    Ajuda a sublinhar o texto incluído na tag <u>.

    É suportada pelo HTML4.1, mas está obsoleta no HTML5.

    Exemplo:

                         
            <u>Conteúdo... </u>
            
    Como aparece em um navagador:
    
             Conteúdo...

    Elemento <i> ou Itálico

    Voltar

    Marque o texto que se destaca em um documento.

    Por exemplo: termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios.

    Exemplo:

                <i>Flamengo</i>, melhor time do Brasil.
        
    Como aparece em um navagador:
    
            Flamengo, melhor time do Brasil.

    Elemento <strong> ou Negrito

    Voltar

    <strong> dá ao texto uma forte importância, é mostrado em negrito.

    Exemplo:

                <p>
            Ao fazer x é <strong>imperativo</strong> que se faça y antes de prosseguir.
            </p>
        
    Como aparece em um navagador:
    
            Ao fazer x é imperativo que se faça y antes de prosseguir.

    Elemento <small>

    Voltar

    é empregado para marcar avisos obrigatórios, restrições legais ou restrições de uso de conteúdo como copyrights,

    ou seja, não é apenas para tornar o tamanho do texto menor.

    Listas

    Voltar

    <li> (Lista dos Itens de um elemento HTML)

    É usado para representar um item que faz parte de uma lista, deve estar contido em um elemento:

    <ol> ou <ul> (lista ordenadas e lista não ordenadas respctivamente)

    Elemento <ol> ou listas ordenadas:

    Voltar

    É também uma lista de itens, são marcadas com números.

    Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>.

    Exemplo:

                <ol>
    
            <li>Café</li>
    
            <li>Leite</li>
    
            </ol>
        
    Como aparece em um navagador:
    
        
            1.Café
    
            2.Leite

    Elemento <ul> ou listas não ordenadas:

    Voltar

    lista não ordenada é uma lista de itens. As listas de itens são marcadas com bullets (pequenos círculos pretos "•").

    Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag <li>.

    Exemplo:

                <ul>
    
            <li>Café</li>
    
            <li>Leite</li>
    
            </ul>
    
        
    Como aparece em um navagador:
    

            • Café
    
            • Leite

    Dentro de um item de uma lista você pode colocar parágrafos, quebras de linha, imagens, vínculos, outras listas, etc.

    Link

    Voltar

    O link é criado envolvendo o texto que você quer transformar em um link dentro de um elemento <a>,

    br> e dando-lhe um atributo href, que conterá o endereço da Web para o qual você deseja que o link aponte.

    O elemento <a> ou Âncora

    Voltar

    com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

    href: (Referência para Hipertexto) utilizada em linguagem HTML que indica o endereço para o qual remete um link.

    Exemplo:

                <p>
                Criando um link para
                <a href="https://www.mozilla.org/pt-BR/"> página inicial da Mozilla</a>.
            </p>

    O atributo target e title

    Voltar

    O atributo target diz ao navegador onde abrir o link.

    Para abrir um link em uma nova aba, apenas defina o atributo target como _blank.

    O atributo title por sua vez é mostrado como uma "dica" quando se passa o mouse sobre o elemento.

    Ele pode ser importante em alguns momentos, como uma espécie de legenda da imagem ou para dar alguma instrução ao usuário.

    Exemplo:

                <p>
                Criando um link para abrir em uma nova aba
                Confira o <a href="https://www.freecodecamp.org/" title="Clique aqui" target="_blank">freeCodeCamp</a>.
            </p>

    Atributo global id

    Voltar

    define um identificador exclusivo (ID) que deve ser único por todo o documento.

    Seu objetivo é identificar o elemento ao navegar por âncoras.

    Exemplo:

                <ul>
    
            <li><a href="#Listas">Listas</li>
    
            </ul>
    
            
            <h2 id="#Listas">Listas</h2>
